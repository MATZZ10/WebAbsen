# Simple project

Stack:
- Next.js App Router
- NextAuth/Auth.js
- Vercel Postgres + SQL langsung
- shadcn-style UI components
- TanStack Table
- Excel/PDF export

## Setup
1. Copy `.env.example` to `.env.local`
2. Create Vercel Postgres / set `DATABASE_URL`
3. Run schema SQL
4. Install dependencies
5. `npm run dev`

## Seed
`npm run seed`
